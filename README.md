# Calculator Frontend Project
This project aims to create a better frontend for a calculator and host it locally.

## How to Run
1. Clone the repository: `git clone https://github.com/faaz0saleem/make-the-frontend-of-calculater-better-a.git`
2. Navigate to the project directory: `cd make-the-frontend-of-calculater-better-a`
3. Open the `index.html` file in a web browser to use the calculator.

## Known edge cases
* The calculator does not handle division by zero.
* The calculator does not support advanced mathematical operations like exponentiation or roots.
